The Flaky Network

Advanced Adaptive Forward Error Correction System

System Architecture

The system is engineered as an ultra-low latency, non-blocking UDP transport layer written in C++17. It proactively combats extreme network jitter, random drops, and burst losses without relying on costly retransmissions.

The receiver operates on a strict Zero-Delay Forwarding policy. Data packets are immediately forwarded to the player upon arrival. Parity recovery happens synchronously via XOR operations, reconstructing lost payloads in microseconds.

The Mathematics of Interleaved XOR

Traditional retransmission (NACK) fails in real-time environments due to the speed-of-light delay traversing the network twice. We solve this mathematically using Forward Error Correction (FEC).

Our specific implementation utilizes an Interleaved XOR Parity (K=2) model. For every two data frames, we generate a single parity packet (1.5x bandwidth overhead, well under the 2.0x limit).

To defeat burst losses, the sender interleaves the parity pairs based on a Stride parameter. If Stride = 2, Frame 0 pairs with Frame 2. If a network outage drops two consecutive packets (Frame 1 and 2), they belong to different parity groups and are fully mathematically recoverable.

Closed-Loop Adaptive Feedback

Higher strides provide immense burst protection but inherently demand a larger delay budget (the sender must wait longer to receive the parity partner). Hardcoding the stride is a guessing game against an unknown network.

We solved this by establishing a Closed-Loop Telemetry Path:

  • The Observer: Every 25 frames (500ms), the Receiver scans the arrival timeline, calculating the exact length of burst losses and overall packet drop volume.
  • The Signal: An 8-byte telemetry packet is beamed back to the Sender via the hostile network.
  • The Adaptation: The Sender dynamically adjusts the XOR Stride on the fly. If the network is calm, it drops to Stride 1 (maximizing jitter headroom). If it detects an active storm, it shifts to Stride 2 to catch the bursts.
